작동 방식

설치하고
클릭 한 번

01

ZIP 다운로드 후 설치

크롬 설정에서 개발자 모드를 켜고 폴더를 드래그하면 끝입니다. 3분이면 충분해요.

02

뉴스 기사 페이지로 이동

네이버, 다음, 조선일보, 한겨레, 뉴시스 — 어떤 언론사 사이트든 동작합니다.

03

🔍 아이콘 클릭

분석 버튼 하나만 누르면 전체 기사에 색상 하이라이트가 즉시 적용됩니다.

🟢

팩트 기반

공식 기관 발표, 수치, 날짜·통계 데이터가 포함된 문장입니다. 실제 진위가 검증됐다는 뜻은 아니며 근거 표현이 있다는 의미입니다.

🟡

의견·주장

전문가 예측, 개인 평가, "~것으로 보인다" 표현 등 주관적 판단이 담긴 문장입니다. 사실이 아닌 견해예요.

🔴

감정 자극

충격·경악·분노 등 감정을 자극하거나, 낙인찍기·극단적 표현으로 판단에 영향을 줄 수 있는 문장입니다.

⬜

중립·맥락

특별한 편향 지표가 없는 배경 설명이나 맥락 문장입니다. 기사 구조를 이해하는 데 도움이 됩니다.

설치 방법

3단계로 설치 완료

1

ZIP 파일 다운로드 & 압축 해제

아래 버튼을 눌러 ZIP을 받고, 원하는 폴더에 압축을 해제하세요.
폴더는 삭제하지 마세요 — 크롬이 항상 이 폴더를 참조합니다.

2

크롬에서 개발자 모드 활성화

크롬 주소창에 chrome://extensions/ 입력 → 오른쪽 상단 "개발자 모드" 토글 ON → "압축 해제된 확장 프로그램 로드" 클릭 → 압축 해제한 폴더 선택

3

뉴스 기사에서 🔍 클릭

크롬 툴바에 🔍 아이콘이 생겼습니다. 뉴스 기사 페이지에서 아이콘을 클릭하고 "이 페이지 분석하기" 버튼을 누르면 완료!
