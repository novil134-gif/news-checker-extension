# 뉴스 신뢰도 판별기 — 크롬 익스텐션

뉴스 기사의 각 문장을 **팩트 / 의견 / 감정조작 / 중립**으로 실시간 분류합니다.

## 📦 설치 방법

### 1단계: 크롬에 익스텐션 로드
1. 크롬 브라우저에서 주소창에 `chrome://extensions/` 입력
2. 오른쪽 상단 **"개발자 모드"** 토글 켜기
3. **"압축 해제된 확장 프로그램 로드"** 클릭
4. 이 폴더(`news-checker-extension`) 선택

### 2단계: API 키 설정
1. 크롬 툴바에서 🔍 아이콘 클릭
2. [Anthropic Console](https://console.anthropic.com/settings/keys)에서 API 키 발급
3. `sk-ant-...` 형태의 키 입력 후 **저장** 클릭

### 3단계: 사용
1. 뉴스 기사 페이지로 이동
2. 🔍 아이콘 클릭 → **"이 페이지 분석하기"** 클릭
3. 10~20초 후 기사에 색상 하이라이트 적용됨
4. 문장 위에 마우스를 올리면 판별 이유 툴팁 표시

## 🎨 색상 의미
| 색상 | 의미 |
|------|------|
| 🟢 초록 밑줄 | 검증 가능한 팩트 (수치, 공식 발표) |
| 🟡 노랑 밑줄 | 전문가·개인의 의견, 예측, 주관적 평가 |
| 🔴 빨강 밑줄 | 감정 조작, 낙인찍기, 공포 유발 표현 |
| ⬜ 회색 밑줄 | 중립적 배경 설명 |

## ⚠️ 주의사항
- Anthropic API 사용 비용이 발생합니다 (기사 1개 분석 ≈ $0.003)
- API 키는 브라우저 로컬 스토리지에만 저장됩니다
- 페이지 구조에 따라 일부 사이트에서 하이라이트 적용이 불완전할 수 있습니다
